# Travel Advisor Web Application 

Get Info about restaurants, hotels and tourist attractions near by in a few clicks. Know better. Book better. Go better. Trusted reviews and photos.

### Technologies Used: 
- ReactJs
- MaterialUI

### Api Used: 
- Google Map Api
- Travel Advisor Rapid Api
